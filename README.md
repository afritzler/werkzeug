# werkzeug

![logo](/docs/images/logo.png)

Opinionated Shell, Vim, Tmux Setup

## Supported Tools

* tmux
* vim
* kubectl
* zsh

## Supported Operating Systems

* Ubuntu
* MacOS

## Installation

You need `bash`, `git` and `curl` to be present on machine in order to install `werkzeug`.

```bash
curl SOMETHING
```
